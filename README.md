## Kemono Helper

A script makes download easier.

+ Get aria2 links for downloadable files
+ Directly download with a file picker, visible download status and retry download when failed.

[Install this script.](https://github.com/mengshitia/userscripts/raw/main/kemono-helper.user.js)

## Youtube videos playback volume fix

Prevent Youtube videos' volume from automatically changing when starting play, unpausing or changing the external volume.

[Install this script.](https://github.com/mengshitia/userscripts/raw/main/youtube-volume.user.js)
